核心对数公式

（条件：a>0 且 a≠1，M>0，N>0）

1. 基本定义

$\log_a N = b \iff a^b = N$ 
2. 运算法则

• \log_a(MN) = \log_a M + \log_a N
• \log_a\frac{M}{N} = \log_a M - \log_a N
• \log_a M^n = n\log_a M（n 为实数）

3. 换底公式

\log_a N = \frac{\log_c N}{\log_c a}（补充条件：c>0 且 c≠1）

4. 特殊值公式

• \log_a 1 = 0
• \log_a a = 1
• a^{\log_a N} = N（对数恒等式）
